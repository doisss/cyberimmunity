# Cyberimmunity Modules

Учебные проекты по курсу **«Кибериммунитет»**: практика проектирования и реализации систем на основе кибериммунного подхода — от простых обменивающихся сообщениями сущностей на MQTT до архитектуры БЛА, интернет-магазина и защищённого мессенджера.

> Модуль №1 и №2 не входят в репозиторий — в них не требовалось программирование.

## Содержание

| № | Модуль | Описание | Стек |
|---|------|----------|------|
| 3 | [Базовая демонстрация кибериммунитета](03-cyberimmunity-basics-demo) | Изучение и модификация модели сущностей и политик безопасности | Python, multiprocessing |
| 4 | [Кибериммунный светофор](04-cyberimmune-traffic-light) | Управление светофором с монитором безопасности, самодиагностикой и внешним коннектором | Python, multiprocessing |
| 5 | [Общительные сущности](05-communicating-entities-mqtt) | Обмен сообщениями между сущностями через брокер Mosquitto (pub/sub) | Python, paho-mqtt |
| 6 | [Киберустойчивый БЛА «Сокол» — архитектура](06-falcon-drone-architecture) | Проектирование архитектуры БЛА для мониторинга дорожного покрытия (PlantUML, DrawIO) | Отчёт (Word) |
| 7 | [БЛА «Сокол» — программная реализация](07-falcon-drone-implementation) | Реализация архитектуры модуля 6: обмен телеметрией через MQTT, детекция аномалий (IQR) | Python, paho-mqtt, numpy |
| 8 | [Кибериммунный магазин](08-cyberimmune-shop) | Онлайн-магазин на Django с защищённой БД, резервным копированием и админ-панелью | Python, Django |
| 9 | [Лучший мессенджер в мире](09-secure-messenger) | Защищённый обмен сообщениями между двумя клиентами через сервер-посредник | Python, sockets, RSA |

Каждая папка содержит:
- `README.md` — задание модуля;
- `imgs/` — схемы, иллюстрирующие задание;
- решение — отдельным файлом (`.py`) или отчётом (`.docx`).
